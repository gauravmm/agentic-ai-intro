# Label Design

## Assessment Questions

After the exercise, ask students:

1. What was your agent's first action? Why?
2. How did your agent handle the missing resources?
  a. Did it ask permission to access the hinted library.
3. Did your agent ask for clarification? When?
4. How did your agent verify its solution worked?
  a. It probably didn't, until you gave it access to the tool.
